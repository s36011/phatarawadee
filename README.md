# 🐾 แฟ้มสะสมผลงานสู่คณะสัตวแพทยศาสตร์ มหาวิทยาลัยมหิดล

เว็บไซต์แฟ้มสะสมผลงานส่วนตัว (Portfolio Website) ที่จัดทำขึ้นเพื่อรวบรวมข้อมูลส่วนตัว ผลงาน กิจกรรม ทักษะ และความมุ่งมั่นในการเตรียมความพร้อมสู่การศึกษาต่อในคณะสัตวแพทยศาสตร์ มหาวิทยาลัยมหิดล

เว็บไซต์นี้พัฒนาด้วย HTML, CSS และ JavaScript พร้อมเผยแพร่ผ่าน GitHub เพื่อแสดงทักษะด้านเทคโนโลยีควบคู่กับความสนใจทางด้านวิทยาศาสตร์สุขภาพและสัตวแพทยศาสตร์

---

* **ระดับชั้น:** มัธยมศึกษาปีที่ 6/4
* **เลขที่:** 36
* **เป้าหมายทางการศึกษา:** คณะสัตวแพทยศาสตร์ มหาวิทยาลัยมหิดล

---

## 🎯 วัตถุประสงค์ของเว็บไซต์

* นำเสนอข้อมูลส่วนตัวและความสนใจทางวิชาการ
* รวบรวมผลงานและกิจกรรมที่เกี่ยวข้องกับสายวิทยาศาสตร์สุขภาพ
* แสดงทักษะด้านเทคโนโลยีและการพัฒนาเว็บไซต์
* ใช้เป็นแฟ้มสะสมผลงานประกอบการศึกษาต่อ

---

## ✨ คุณสมบัติของเว็บไซต์

* Responsive Design รองรับการแสดงผลบนคอมพิวเตอร์ แท็บเล็ต และโทรศัพท์มือถือ
* พื้นหลังแบบ Animated Gradient ด้วย CSS Animation
* ส่วนแสดงข้อมูลส่วนตัว ความฝัน และเป้าหมายทางการศึกษา
* ส่วนแสดงกิจกรรมและผลงานที่เกี่ยวข้องกับสายสัตวแพทยศาสตร์
* ระบบแสดงปีลิขสิทธิ์ปัจจุบันอัตโนมัติด้วย JavaScript
* ออกแบบด้วยธีมสีชมพูสดใส พร้อมเอฟเฟกต์ Hover และ Animation

## 🐾 กิจกรรมและผลงานที่ภาคภูมิใจ

### 🩺 ค่ายเจาะลึกสัตวแพทย์ (Veterinary Camp)

เข้าร่วมกิจกรรมเรียนรู้ทักษะวิชาชีพสัตวแพทย์เบื้องต้น ฝึกจำลองการตรวจร่างกายสัตว์ และศึกษากายวิภาคศาสตร์ของสัตว์

### 🔬 โครงงานวิทยาศาสตร์

ศึกษาและทำการทดลองด้านชีววิทยา เพื่อพัฒนาทักษะการคิดวิเคราะห์และกระบวนการทางวิทยาศาสตร์

### 💻 เว็บไซต์ Portfolio ส่วนตัว

ฝึกทักษะการพัฒนาเว็บไซต์ด้วย HTML, CSS และ JavaScript พร้อมเรียนรู้การใช้งาน Git และ GitHub
